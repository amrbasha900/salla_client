# MCP Usage Guide

This project keeps Salla documentation links in `docs/llms.txt`. Always reference that file
when you need official documentation URLs.

## Primary sources
- `docs/llms.txt` (canonical Salla docs links)
- `docs/openapi/salla_merchant_api.yaml` (Salla Merchant API minimal spec)
- `docs/CONTRACT.md` (Phase 2 command protocol)

## Guidance
- Prefer links from `docs/llms.txt` when documenting Salla endpoints.
- Keep OpenAPI additions in `docs/openapi/salla_merchant_api.yaml`.
- Update `docs/CONTRACT.md` as Phase 2 interfaces evolve.
